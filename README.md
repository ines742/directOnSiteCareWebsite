# Skinny Drip Website 

A responsive single-page application for IV thereapy, Esthetics and Weight loss services. Built with Vite + React and Tailwind CSS v4, it focuses on fast performance, clean semantics, and a mobile-first UI.

## Technologies
- React 19, React Router
- Vite
- Tailwind CSS v4
- Font Awesome

## Features
- Responsive, mobile‑first UI with Tailwind v4
- Client‑side routing (Home, Esthetics, IV Therapy, Weight Loss, About, Privacy, Booking)
- Reusable components (Navbar, Footer, Button)
- Semantic HTML (section, article, header/footer)
- Accessible images (contextual alt) and keyboard‑friendly nav
- Branded gradients and typography via custom utility classes

## Development Process
- Styling: Tailwind utilities plus custom classes in @layer utilities (e.g., .h1, .h2, .base-bold, gradient .g1–.g6).
- Routing: React Router; Navbar links navigate between pages.
- Assets: Public assets referenced by absolute paths (/images/…), or imported from src/assets.
- Accessibility: Alt text in components; headings and sections for structure.

## Running the project
1. Clone the repository
2. Install dependencies: npm install
3. Start dev server: npm run dev
4. Open the URL Vite prints
